# VWX MCP Server — Vectorworks 2024

Local Codex bridge for Vectorworks 2024. Create a Python menu command in **Tools > Plug-ins > Plug-in Manager**, paste `vectorworks/VWX MCP Bridge Process Queue.py`, and run it whenever Codex sends a command. It uses only local files in `Documents/Vectorworks/VWX-MCP-Bridge` and does not execute arbitrary AI scripts.
